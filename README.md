# Better Joker Collection
By Timooo

This Balatro mod allows sorting your joker collection by alphabet, stickers and rarity.
Sorting can happen by ascending and descending order and locked/undiscovered jokers can be ignored.

# Additional Notes:

[Steamodded](https://github.com/Steamodded/smods/releases/latest) is required
